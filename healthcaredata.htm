To save or download a graph: Using your browser,
Right click the graph to download
Select SAVE PICTURE AS ... or SAVE IMAGE AS ...
Select SAVE

To save or download the tabulated results: Using your browser,
Select FILE
Select SAVE AS
Change the suffix of the FILE NAME to HTM (e.g., "MEPSnetIC.jsp to MEPSnetIC.htm")
Select SAVE
Then you can open the saved HTM file in most current word processing or spreadsheet packages.
